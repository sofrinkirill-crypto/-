# Генератор расписания PBO

## Запуск

### 1. Установить зависимости

```bash
pip install -r requirements.txt
```

### 2. Запустить сервер

```bash
python app.py
```

### 3. Открыть в браузере

```
http://localhost:5000
```

---

## Структура проекта

```
scheduler/
├── app.py              # Flask-сервер + алгоритм генерации
├── requirements.txt    # Зависимости (flask)
├── data.json           # Данные сотрудников и запросов (создаётся автоматически)
└── templates/
    └── index.html      # Веб-интерфейс
```

## Уровни сотрудников

| Уровень | Роль | Смены |
|---------|------|-------|
| level5 | Руководители отделов (canInside=True) | mI×2 + eI×2 + rdm×1 = 5 смен |
| level4 | Ассистенты директора (canInside=True) | mI×2 + eI×2 + rdm×1 = 5 смен |
| level3 | Менеджеры / SW | morningManager + eveningManager + sw |
| level6 | Директор | office, сб+вс — выходные (раз в месяц) |

## Правила алгоритма

- После ночной смены — следующий день выходной
- Вечер→утро в тот же день запрещено
- Не более 2 ночных смен в неделю
- level6 получает сб+вс выходными один раз в месяц
- Запросы на выходные учитываются при составлении расписания
